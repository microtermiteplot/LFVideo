# Ideas（头脑风暴 / 选题参考库）

> 这里放**未定稿**的脑暴内容：候选选题、每期参考素材、灵感碎片。可随意改、随意删。

## 与 content-library 的边界

| | ideas/ | content-library/ |
|---|--------|------------------|
| **内容状态** | 未定稿、可随意改 | 人工审核后的定稿 |
| **用途** | 想清楚做什么 | 已经生产好、可发布 |
| **进入下一步** | 选题确定后 → 走 workflow 01 | 看板标 approved → 进下一阶段 |

## 文件说明

- `backlog.md` — 候选选题池（一句话清单，未排期）
- `epNN-slug.md` — 某期的脑暴参考素材（核心卖点、失败展示候选、灵感）

## 流转规则

脑暴成熟 → 在 `backlog.md` 排期 → 执行 `/01-topic-research` 产出选题分析 → 落盘到 `content-library/epNN-slug/01-topic.md`。
