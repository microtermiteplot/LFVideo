# ep01: AI IDE 的三层认知框架（脑暴参考）

**选题方向**: 概念科普 / A 轨纯自动化

## 核心卖点

- **Editor 模式**: 你主导，AI 执行单步
- **Agent 模式**: 给目标，AI 自动拆解执行
- **Role 模式**: 给身份边界，AI 在约束下自主工作

## 失败展示候选

- Agent 模式无限循环卡住案例
- Role 模式角色边界模糊导致输出偏离预期

## 技术备注

A 轨试跑验证通过，Remotion 模板库可用
